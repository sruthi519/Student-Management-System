# StudentManagementSystem
Student Management System is a DBMS-based web application developed using PHP, HTML, CSS, and MySQL. It enables efficient management of student records, courses, faculty details, attendance, and marks through a simple and user-friendly interface.
Features
Add Student Details
Store Student Name and Email
Add Course Information
Manage Faculty Records
Record Student Attendance
Store Student Marks
MySQL Database Integration
Simple and User-Friendly Interface

PHP
HTML

Apache
MySQL

studentdb
Step 3

Import the SQL file containing the required tables.

Step 4

Place the project folder inside:

xampp/htdocs/
Step 5

Open the browser and run:

http://localhost/StudentManagementSystem
